Data cleaning project

A structured SQL project focused on transforming messy, inconsistent data into a clean and reliable foundation for analysis.

Retail Sales Dashboard: Deep Dive

This project focuses on transforming raw, inconsistent data into a reliable, analysis ready dataset using advanced SQL techniques. The goal was to build a foundation suitable for reporting and visualization. Key features include:

• Country Field Standardization

Identified and replaced inconsistent country name variants (e.g., 'US', 'us', 'United States') with a clean 'USA' value using conditional UPDATE statements.

• Birth Date Format Correction

Cleaned the date_of_birth column by parsing multiple text based formats such as 'DD-MM-YYYY', 'YYYY/MM/DD', and 'YYYY-MM-DD'. Implemented a robust CASE block using STR_TO_DATE() to convert entries into valid SQL DATE format.

• Null and Blank Value Handling

Used logical checks like IS NULL and TRIM() to identify fields that were either empty or padded with whitespace. Replaced blank or missing values with placeholders such as 'Unknown’.

• Clean Column Creation

Added new columns (e.g., clean_dob) to preserve the original data and store cleaned outputs.
